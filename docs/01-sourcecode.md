# Mainframe Data Utilities V2

## Downloading the code

### Clone this repo:

```
git clone -b mdu_v2 git@github.com:aws-samples/mainframe-data-utilities.git.
```

### Change directory:

```
cd mainframe-data-utilities/src
```
